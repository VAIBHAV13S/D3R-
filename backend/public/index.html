Block-Donate API Server

Welcome to the Block-Donate API server. This server provides backend functionality for the Block-Donate DApp.

Available Endpoints

GET /api/contract-addresses

Returns the addresses of deployed smart contracts.

POST /api/ngo/register

Register a new NGO with verification documents.

POST /api/milestone/proof

Submit proof for a project milestone.

POST /api/disaster/register

Register a new disaster with evidence.

GET /api/disaster/verification/:disasterId

Get verification status for a disaster.

POST /api/ipfs/upload

Upload a file to IPFS.

GET /api/ipfs/status

Check if IPFS service is available.

Frontend Connection

The frontend DApp connects to this API server to perform blockchain operations and IPFS uploads.

Make sure the frontend is configured with the correct API URL (default: http://localhost:3000).
